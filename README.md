# Horror of Bhangadh — Interactive 3D Experience

## 🧠 Core Philosophy

> “Don’t build sections. Build a world.”

- Scroll = camera movement
- UI = part of the 3D environment

---

## 🚀 Tech Stack

- **Framework:** React 19 + TypeScript
- **Bundler:** Vite 8
- **3D Engine:** Three.js + React Three Fiber + Drei
- **Physics (planned):** React Three Rapier
- **Animations:** GSAP + ScrollTrigger
- **Styling:** Tailwind CSS 4
- **Shaders:** GLSL (vite-plugin-glsl)
- **State Management (planned):** Zustand

---

## 🛠️ Getting Started

### Prerequisites

- Node.js (LTS recommended)
- pnpm / npm / yarn

### Installation

```bash
git clone [repository-url]
pnpm install
```

### Development

```bash
pnpm dev
```

### Production Build

```bash
pnpm build
```
